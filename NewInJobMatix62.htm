JobMatix JobTracking provides proven reliable job-tracking and RMA Tracking (Returns to Supplier) functionality for computer service shops, and similar businesses. JobMatix JobTracking is a support product to both JobMatixPOS and MYOB Retail Manager POS systems. For all customer, stock, sales quotes etc information, JobMatix refers to the relevant retail data tables. Where MYOB is used, Jobmatix accesses the Retail Manager Jet database in read-only mode only.

JobMatixPOS

JobMatixPOS is a Retail-POS desktop/SQL system designed as a migration path from MYOB Retail Manager, and as an alternative retail/stock partner host for JobMatix JobTracking.

Retail POS functions supported include Suppliers and Purchasing, Stock Control and Goods Received, serials tracking, customer account and cash sales, laybys, cash and EftPos payment types, Account payments and Debtors statements.. And now also has Customer Subscriptions functionality to support Rentals and Managed Services.

Existing Retail-Manager and JobMatix JobTracking users will be able to migrate the MYOB-RM database information from their staff, suppliers, stock (incl. serials), and customer tables ( ie from the recent.mdb database) to JobMatixPOS using the migration feature. (NB: RM Sales Invoices and Payments Received are not migrated).

For JobMatix POS users, JobTracking job delivery/sale would thereafter be invoiced directly via the POS Sale screen, bypassing the need to scan in the list of job parts (barcodes) into the sale.

(Note: The POS functionality in JobMatixPOS dll 4.2.4287 is at late Beta stage, and not yet in full release mode. Database structure changes may still be made.. Users should test JobMatixPOS fully in trial mode, before using it in a live production environment..)
